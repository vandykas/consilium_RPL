# Consilium

Consilium adalah aplikasi web yang dirancang untuk mempermudah proses bimbingan Tugas Akhir (TA) antara Dosen dan Mahasiswa di lingkungan Program Studi Informatika Universitas Katolik Parahyangan. Aplikasi ini menyediakan sistem penjadwalan bimbingan, manajemen akun, notifikasi, serta pencatatan hasil bimbingan secara digital dan terstruktur.
